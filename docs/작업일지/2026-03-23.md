## Stage 5 설계: 합성 품질 점수

CASDA `score_casda_quality.py`의 3-metric을 그대로 적용.

### 평가 지표 (가중치)

| 지표 | 가중치 | 설명 |
|---|---|---|
| `color_score` | 0.40 | LAB 색공간 기준 배경-패치 색상 일관성 |
| `artifact_score` | 0.30 | Gradient 기반 합성 경계 아티팩트 감지 |
| `blur_score` | 0.30 | Laplacian + Gradient 선명도 |

### 출력

- `stage4_output/{seed_id}/quality_scores.json`
  ```json
  [
    {
      "image_id": "000",
      "color_score": 0.82,
      "artifact_score": 0.74,
      "blur_score": 0.91,
      "quality_score": 0.82
    }
  ]
  ```

### 구현 대상 파일

- `utils/quality_scoring.py` — `compute_quality_score(bg, composite, mask)` 신규 작성
- `stage5_quality_scoring.py` — 배치 실행 스크립트

## Stage 6 설계: 증강셋 구성

CASDA `package_casda_data.py` + `merge_datasets.py` 대응.

### Dataset Groups (CASDA 대응)

| Group | 내용 | CASDA 대응 |
|---|---|---|
| `baseline` | `train/good` 원본만 | `baseline_raw` |
| `aroma_full` | 원본 + 전체 stage4 합성 | `casda_composed` |
| `aroma_pruned` | 원본 + quality_score 필터링 합성 | `casda_composed_pruning` |

### 출력 구조 (MVTec-style)

```
{cat_dir}/augmented_dataset/
├── {group}/
│   ├── train/
│   │   ├── good/        ← 원본 train/good
│   │   └── defect/      ← stage4_output defect 이미지 (aroma_* 그룹만)
│   └── test/
│       ├── good/
│       └── {defect_type}/
```

### 필터링 기준

- `quality_score >= 0.6` → `aroma_pruned` 포함
- 임계값은 Step 11-3 점수 분포 확인 후 조정

### 구현 대상 파일

- `stage6_dataset_builder.py` — dataset_groups별 디렉토리 구성

## Stage 7 설계: 모델 벤치마크

CASDA `run_benchmark.py` 대응. AROMA는 이상 탐지(AD) 모델 사용.

### 벤치마크 모델

| 모델 | 유형 | 선택 이유 |
|---|---|---|
| **PatchCore** | Memory-based AD | MVTec AD SOTA, 사실상 표준 baseline |
| **PaDiM** | Distribution-based AD | 빠른 학습, 도메인 일반화 검증용 |
| **EfficientAD** | Student-Teacher AD | 경량 + 높은 성능, 실용성 검증용 |

### 평가 지표

| 지표 | 설명 |
|---|---|
| **Image-AUROC** | 이미지 레벨 이상 탐지 성능 |
| **Pixel-AUROC** | 픽셀 레벨 이상 위치 탐지 성능 |
| **PRO** | Per-Region Overlap (지역 단위 정밀도) |

### Dataset Groups × 모델 실험 매트릭스

```
3 models × 3 dataset groups × 3 domains = 27 실험
```

| | baseline | aroma_full | aroma_pruned |
|---|---|---|---|
| PatchCore | ✓ | ✓ | ✓ |
| PaDiM | ✓ | ✓ | ✓ |
| EfficientAD | ✓ | ✓ | ✓ |

### Inject/Clean 전략 (CASDA 동일)

- `baseline` 데이터에 증강 이미지 inject → 학습 → clean (공간 절약)
- 자동 재개 지원 (`--resume`)

### 구현 대상 파일

- `configs/benchmark_experiment.yaml` — 실험 설정
- `stage7_benchmark.py` — 벤치마크 오케스트레이터
- `utils/ad_metrics.py` — AUROC/AUPR/PRO 계산
- `scripts/analyze_results.py` — 결과 분석 + 비교표

```python
import json, sys
from pathlib import Path
from tqdm.auto import tqdm

sys.path.insert(0, "/content/aroma")
from stage5_quality_scoring import run_quality_scoring

REPO   = Path("/content/drive/MyDrive/project/aroma")
CONFIG = json.loads((REPO / "dataset_config.json").read_text())

DOMAIN_FILTER = "visa"   # "isp" / "mvtec" / "visa"
LABEL = {"isp": "ISP-AD", "mvtec": "MVTec AD", "visa": "VisA"}[DOMAIN_FILTER]

ENTRIES = [(k, v) for k, v in CONFIG.items()
           if not k.startswith("_") and v["domain"] == DOMAIN_FILTER]

all_seeds, skip = [], 0
for key, entry in ENTRIES:
    cat_dir    = Path(entry["seed_dir"]).parents[1]
    stage4_dir = cat_dir / "stage4_output"
    if not stage4_dir.exists():
        continue
    for d in stage4_dir.iterdir():
        if not d.is_dir():
            continue
        if (d / "quality_scores.json").exists():
            skip += 1
        else:
            all_seeds.append((cat_dir, d.name))

if not all_seeds:
    print(f"✓ {LABEL} 모든 작업 완료")
else:
    print(f"{LABEL}: {len(all_seeds)} seeds 처리 예정 (skip {skip}건)")
    failed = []
    for cat_dir, seed_id in tqdm(all_seeds, desc=f"Stage5 {LABEL}"):
        try:
            run_quality_scoring(
                stage4_seed_dir = str(cat_dir / "stage4_output" / seed_id),
                workers = 0,
            )
        except Exception as e:
            failed.append({"category": cat_dir.name, "seed": seed_id,
                           "error": str(e), "type": type(e).__name__})
    print("\n" + (f"✓ {LABEL} 완료" if not failed else f"✗ {len(failed)}건 실패"))
    for f in failed:
        print(f"  [{f['category']}/{f['seed']}] {f['type']}: {f['error'][:100]}")
```

### Step 11-3: 점수 분포 검증

```python
import json
from pathlib import Path
import numpy as np

REPO   = Path("/content/drive/MyDrive/project/aroma")
CONFIG = json.loads((REPO / "dataset_config.json").read_text())

all_scores = []
for key, entry in CONFIG.items():
    if key.startswith("_"):
        continue
    cat_dir    = Path(entry["seed_dir"]).parents[1]
    stage4_dir = cat_dir / "stage4_output"
    if not stage4_dir.exists():
        continue
    for d in stage4_dir.iterdir():
        cache = d / "quality_scores.json"
        if not cache.exists():
            continue
        data = json.loads(cache.read_text())
        all_scores.extend(s["quality_score"] for s in data.get("scores", []))

arr = np.array(all_scores)
print(f"전체 이미지 수: {len(arr)}")
print(f"mean={np.mean(arr):.4f}  std={np.std(arr):.4f}")
print(f"P25={np.percentile(arr,25):.4f}  P50={np.percentile(arr,50):.4f}"
      f"  P75={np.percentile(arr,75):.4f}  P90={np.percentile(arr,90):.4f}")
print(f"\nPruning threshold 후보:")
print(f"  P25 기준 ({np.percentile(arr,25):.3f}): 상위 75% 선택")
print(f"  P50 기준 ({np.percentile(arr,50):.3f}): 상위 50% 선택")
print(f"  P75 기준 ({np.percentile(arr,75):.3f}): 상위 25% 선택 (aggressive)")
```

### Step 12-2a/b/c: 배치 데이터셋 구성

```python
import json, sys
from pathlib import Path
from collections import defaultdict
from concurrent.futures import ThreadPoolExecutor, as_completed
from tqdm.auto import tqdm

sys.path.insert(0, "/content/aroma")
from stage6_dataset_builder import run_dataset_builder

REPO   = Path("/content/drive/MyDrive/project/aroma")
CONFIG = json.loads((REPO / "dataset_config.json").read_text(encoding="utf-8"))

DOMAIN_FILTER = "visa"   # "isp" / "mvtec" / "visa"
LABEL = {"isp": "ISP-AD", "mvtec": "MVTec AD", "visa": "VisA"}[DOMAIN_FILTER]

# Google Drive I/O 병렬도 설정
NUM_IO_THREADS = 8  # 카테고리 내부 파일 복사 스레드 수 (I/O-bound → Thread 유리)
CAT_THREADS    = 2  # 카테고리 단위 동시 처리 수 (Drive 동시 쓰기 안정성 고려)

cat_map: dict[str, str] = {}
cat_seed_dirs: dict[str, list] = defaultdict(list)
for key, entry in CONFIG.items():
    if key.startswith("_") or entry["domain"] != DOMAIN_FILTER:
        continue
    cat_dir   = str(Path(entry["seed_dir"]).parents[1])
    image_dir = entry["image_dir"]
    if cat_dir in cat_map and cat_map[cat_dir] != image_dir:
        raise ValueError(f"image_dir 불일치: {cat_dir}")
    cat_map[cat_dir] = image_dir
    cat_seed_dirs[cat_dir].append(entry["seed_dir"])

all_cats, skip = [], 0
for cat_dir, image_dir in cat_map.items():
    if (Path(cat_dir) / "augmented_dataset" / "build_report.json").exists():
        skip += 1
    else:
        all_cats.append((cat_dir, image_dir, cat_seed_dirs[cat_dir]))

if not all_cats:
    print(f"✓ {LABEL} 모든 작업 완료")
else:
    print(f"{LABEL}: {len(all_cats)} categories 처리 예정 (skip {skip}건)")
    failed = []

    def _run_one(cat_dir, image_dir, seed_dirs):
        run_dataset_builder(cat_dir=cat_dir, image_dir=image_dir,
                            seed_dirs=seed_dirs, workers=NUM_IO_THREADS)

    with tqdm(total=len(all_cats), desc=f"Stage6 {LABEL}") as bar:
        with ThreadPoolExecutor(max_workers=CAT_THREADS) as ex:
            futs = {ex.submit(_run_one, c, i, s): (c, i, s)
                    for c, i, s in all_cats}
            for fut in as_completed(futs):
                cat_dir, image_dir, seed_dirs = futs[fut]
                try:
                    fut.result()
                except Exception as e:
                    failed.append({"category": Path(cat_dir).name,
                                   "error": str(e), "type": type(e).__name__})
                bar.update(1)

    print("\n" + (f"✓ {LABEL} 완료" if not failed else f"✗ {len(failed)}건 실패"))
    for f in failed:
        print(f"  [{f['category']}] {f['type']}: {f['error'][:100]}")
```

*(Step 12-2a: `DOMAIN_FILTER = "isp"`, Step 12-2b: `"mvtec"`, Step 12-2c: `"visa"`)*

### Step 12-4: augmented_dataset zip 패키징 (Drive 저장, 최초 1회)

> Stage 6 완료 후 1회만 실행. 벤치마크 학습 시 Drive 직접 읽기 대신
> zip 단일 파일 복사 → 로컬 해제 방식으로 I/O 병목을 제거한다.

```python
import json, subprocess, sys
from pathlib import Path
from tqdm.auto import tqdm

sys.path.insert(0, "/content/aroma")

REPO   = Path("/content/drive/MyDrive/project/aroma")
CONFIG = json.loads((REPO / "dataset_config.json").read_text(encoding="utf-8"))

# 전체 도메인 cat_dirs 수집
cat_dirs: set[str] = set()
for k, v in CONFIG.items():
    if not k.startswith("_"):
        cat_dirs.add(str(Path(v["seed_dir"]).parents[1]))

skip, done, failed = 0, 0, []
for cat_dir in tqdm(sorted(cat_dirs), desc="Zip packaging"):
    aug_dir  = Path(cat_dir) / "augmented_dataset"
    zip_path = Path(cat_dir) / "augmented_dataset.zip"

    if zip_path.exists():
        skip += 1
        continue
    if not aug_dir.exists():
        failed.append({"cat": Path(cat_dir).name, "reason": "augmented_dataset 없음"})
        continue

    try:
        # ZIP_STORED (-0): PNG는 압축 이득 없음 — 패키징만, CPU 낭비 없음
        # cwd=cat_dir 에서 상대경로 "augmented_dataset" 지정 → 구조 보존
        subprocess.run(
            ["zip", "-r", "-0", str(zip_path), "augmented_dataset"],
            cwd=str(cat_dir),
            capture_output=True,
            check=True,
        )
        done += 1
    except subprocess.CalledProcessError as e:
        failed.append({"cat": Path(cat_dir).name, "reason": e.stderr.decode()[:100]})

print(f"\n✓ 완료 {done}건  |  skip {skip}건  |  실패 {len(failed)}건")
for f in failed:
    print(f"  [{f['cat']}] {f['reason']}")
```

```python
    zip 1개를 /content/로 복사 후 unzip. 이미 해제된 경우 skip.

    Returns:
        로컬 cat_dir 경로 — run_benchmark 에 그대로 전달 가능.

    Raises:
        FileNotFoundError: augmented_dataset.zip 없음 (Step 12-4 미실행).
    """
    cat_name  = Path(cat_dir).name
    local_cat = LOCAL_ROOT / cat_name
    local_aug = local_cat / "augmented_dataset"

    if local_aug.exists():
        print(f"  [{cat_name}] 캐시 hit — skip")
        return str(local_cat)

    zip_src = Path(cat_dir) / "augmented_dataset.zip"
    if not zip_src.exists():
        raise FileNotFoundError(
            f"zip 없음: {zip_src}\n"
            "Step 12-4 (zip 패키징) 을 먼저 실행하세요."
        )

    local_cat.mkdir(parents=True, exist_ok=True)
    dst_zip = local_cat / "augmented_dataset.zip"

    print(f"  [{cat_name}] zip 복사...", end=" ", flush=True)
    shutil.copy2(str(zip_src), str(dst_zip))

    print("압축 해제...", end=" ", flush=True)
    subprocess.run(
        ["unzip", "-q", str(dst_zip), "-d", str(local_cat)],
        check=True,
    )
    dst_zip.unlink()   # zip 삭제 (공간 절약)
    print("완료")
    return str(local_cat)


def cleanup_local(cat_dir: str) -> None:
    """로컬 캐시 삭제 — 다음 카테고리를 위한 /content/ 공간 확보."""
    local_cat = LOCAL_ROOT / Path(cat_dir).name
    if local_cat.exists():
        shutil.rmtree(str(local_cat))


print("✓ 로컬 캐시 유틸 준비 완료")
print(f"  LOCAL_ROOT      : {LOCAL_ROOT}")
print(f"  BENCH_OUTPUT_DIR: {BENCH_OUTPUT_DIR}")
```

### Step 13-3a/b/c: 도메인별 배치 벤치마크

> Step 13-0 실행 후 사용.
> 카테고리별: Drive zip 복사 → /content/ 해제 → 학습(로컬) → Drive 결과 저장 → 로컬 삭제.

```python
import json, sys, yaml
from pathlib import Path
from tqdm.auto import tqdm

sys.path.insert(0, "/content/aroma")
from stage7_benchmark import run_benchmark
# Step 13-0에서 정의한 prepare_local / cleanup_local / BENCH_OUTPUT_DIR 사용

REPO        = Path("/content/drive/MyDrive/project/aroma")
CONFIG      = json.loads((REPO / "dataset_config.json").read_text(encoding="utf-8"))
CONFIG_PATH = str(REPO / "configs/benchmark_experiment.yaml")

# category_filter 로드 — 중복 카테고리 제외
_bench_cfg = yaml.safe_load((REPO / "configs/benchmark_experiment.yaml").read_text())
_excluded  = _bench_cfg.get("category_filter", {}).get("exclude", {})

DOMAIN_FILTER = "visa"   # "isp" / "mvtec" / "visa"
LABEL = {"isp": "ISP-AD", "mvtec": "MVTec AD", "visa": "VisA"}[DOMAIN_FILTER]

MODELS = ["efficientnet_b4", "resnet50", "draem"]
GROUPS = ["baseline", "aroma_full", "aroma_pruned"]

cat_map: dict[str, str] = {}
for key, entry in CONFIG.items():
    if key.startswith("_") or entry["domain"] != DOMAIN_FILTER:
        continue
    cat_dir  = str(Path(entry["seed_dir"]).parents[1])
    cat_name = Path(cat_dir).name
    if cat_name in _excluded.get(DOMAIN_FILTER, []):
        continue
    cat_map[cat_dir] = entry["image_dir"]

# resume: Drive 결과 기준으로 완료 여부 판단
out_base = Path(BENCH_OUTPUT_DIR)
all_cats, skip = [], 0
for cat_dir in cat_map:
    all_done = all(
        (out_base / Path(cat_dir).name / m / g / "experiment_meta.json").exists()
        for m in MODELS for g in GROUPS
    )
    if all_done:
        skip += 1
    else:
        all_cats.append(cat_dir)

if not all_cats:
    print(f"✓ {LABEL} 모든 실험 완료")
else:
    print(f"{LABEL}: {len(all_cats)} categories 처리 예정 (skip {skip}건)")
    failed = []
    for cat_dir in tqdm(all_cats, desc=f"Stage7 {LABEL}"):
        local_cat_dir = None
        try:
            local_cat_dir = prepare_local(cat_dir)          # Drive zip → /content/
            run_benchmark(
                config_path=CONFIG_PATH,
                cat_dir=local_cat_dir,                      # 로컬 경로로 학습
                models=MODELS,
                groups=GROUPS,
                output_dir=BENCH_OUTPUT_DIR,                # 결과는 Drive에 저장
            )
        except Exception as e:
            failed.append({"category": Path(cat_dir).name,
                           "error": str(e), "type": type(e).__name__})
        finally:
            if local_cat_dir:
                cleanup_local(cat_dir)                      # 로컬 캐시 삭제
    print("\n" + (f"✓ {LABEL} 완료" if not failed else f"✗ {len(failed)}건 실패"))
    for f in failed:
        print(f"  [{f['category']}] {f['type']}: {f['error'][:100]}")
```

```python
import json, sys
from collections import defaultdict
from pathlib import Path

sys.path.insert(0, "/content/aroma")
from scripts.analyze_results import (
    collect_results, build_comparison_table,
    save_summary_json, save_comparison_markdown,
    save_comparison_csv, print_improvement,

# Step 13-0에서 정의한 BENCH_OUTPUT_DIR / REPO 사용

output_root = Path(BENCH_OUTPUT_DIR)

# ── 완료 현황 ────────────────────────────────────────────────────────────
MODELS = ["efficientnet_b4", "resnet50", "draem"]
GROUPS = ["baseline", "aroma_full", "aroma_pruned"]

total = done = 0
incomplete = []
for cat_dir in sorted(output_root.iterdir()):
    if not cat_dir.is_dir():
        continue
    for m in MODELS:
        for g in GROUPS:
            total += 1
            if (cat_dir / m / g / "experiment_meta.json").exists():
                done += 1
            else:
                incomplete.append(f"{cat_dir.name}/{m}/{g}")

print_improvement(summary)

CONFIG   = json.loads((REPO / "dataset_config.json").read_text(encoding="utf-8"))
_bench   = __import__("yaml").safe_load((REPO / "configs/benchmark_experiment.yaml").read_text())

cat_domain: dict[str, str] = {}

md_path = output_root / "comparison_table.md"

    print(md_path.read_text(encoding="utf-8"))
```
