    'mvtec_cable': {'linearity':7,'solidity':3,'extent':6,'aspect_ratio':4,'eccentricity':7,'circularity':12},

    total = 0
    for feat in features:

print('-' * 60)
for ds in datasets:

🤖 **Claude**

매우 좋습니다. **Neff = e^H** (Hill diversity index, Jost 2006) — 생태학에서 검증된 지표.

**장점:**

- 균등 분포 시 Neff = K (cable 8종 균등 → Neff=8.0)
- 불균형 시 Neff < K (cable 1종 지배 → Neff→1)
- 연속값 → anchor normalization과 자연스럽게 결합
- Reviewer 질문에 "effective number of classes accounting for distributional evenness" 로 답변 가능

**예상값 (균등 가정):**

| Dataset | K | H = ln(K) | Neff = e^H |
|--|--|--|--|
| cable | 8 | 2.079 | **8.0** |
| isp | 2 | 0.693 | **2.0** |
| cashew | 1 | 0.0 | **1.0** |

anchor = max(Neff) × 1.2 = 8.0 × 1.2 = **9.6** → expected_range `[1.0, 9.6]`

**구현:**
```python
def _class_diversity_neff(morph_rows: List[Dict]) -> float:
    counts = Counter(r.get("defect_type") or "unknown" for r in morph_rows)
    total = sum(counts.values())
    if total == 0:
        return 1.0
    probs = [c / total for c in counts.values()]
    H = -sum(p * math.log(p) for p in probs if p > 0)  # nats
    return math.exp(H)
```

|--|--|--|
| Entropy | 클러스터 label entropy | 유지 |
| ValleyCount | per-class mean (소샘플 문제) | **pooled 복귀**, range `[0.0, 41.0]` |
| ClusterCount | GMM BIC 선택 k | **→ ClassDiversity (Neff = e^H)** |
| InvSilhouette | 1 - silhouette | 유지 (ClusterRecoverability로 개명) |

**ClassDiversity 계산:**
```python
# defect_type 분포의 Shannon entropy → effective number of classes
H = -Σ p_k * ln(p_k)
Neff = exp(H)
# cable(8종)≈8.0, isp(2종)≈2.0, cashew(1종)=1.0
```
range: `[1.0, 9.6]`  
논문 설명: *"Effective number of defect categories estimated from the Shannon entropy of class distributions"*
