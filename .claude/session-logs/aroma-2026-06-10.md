**문제 식별: 과소 bins 대비 bin 수 과다**

92 샘플에 50 bins → **평균 1.84 샘플/bin**.  
빈 bin 다수 발생 → inverted histogram에 잡음 peak → 가짜 valley.

```
sparse 구간 예시:
  bin counts: [0, 2, 0, 3, 0, 2, 0]
  inverted:   [6, 4, 6, 3, 6, 4, 6]

  prominence = 6 - 3 = 3  >  threshold 2.71  → valley 감지!
```

```

```
